# BS Computer Science Academic Map

This project is an interactive **Academic Map** for the Bachelor of Science in Computer Science program. It provides a four-year plan with clickable courses that display detailed course descriptions.

---

## Features

- **Four-Year Academic Map**: Organized by year and semester (Fall/Spring).  
- **Clickable Courses**: Each course links to a detailed description page.  
- **Dynamic Course Descriptions**: Fetches course information from a JSON file.  
- **Course Details Displayed**: Includes title, description, credit hours, lecture/lab contact hours, prerequisites, repeatability, TCCNS equivalent, additional fees, and notes.  
- **Responsive Design**: Works on desktop and mobile devices.  
- **Styled with CSS**: Consistent layout and color scheme.

---

## Technologies Used

- **HTML5** – Structure of pages (`index.html`, `course-description.html`).  
- **CSS3** – Styling (`styles.css`) with variables and responsive design.  
- **JavaScript** – Functionality (`app.js`, `course-description.js`) for dynamic interactivity.  
- **JSON** – Course data stored in `coursedata.json` for easy updates.
